Minimal JSON Example

This page demonstrates an absolute bare minimum JSON configuration file. The only part of the JSON file which exists in this demo is the array containing the timeline node data. If you look at the JSON below, you'll notice that this .json file doesn't even have field names. There's no meta data (e.g. timeline name or last updated date), no JSON field names, and no other settings or options. The timelines node items themselves contain just 3 fields: date, heading, and content. Of course, if you use this method to pass JSON data, your timeline will have minimal styling and functionality and use only default settings.

Note: the outer container with the rounded, black border was added for the demo. This will not appear on your timeline.

HTML Code

<div class="timeline" data-json-config="assets/data/simple-data-array.json">
  <div class="timeline__wrap">
    <div class="timeline__items"></div>
  </div>
</div>
