Getting Started

Small examples to get started quickly. Use the CDN snippet shown below for production, or the local `dist/` files for development.

1. Basic HTML timeline (no JS shown)

Drop this HTML into your page. The timeline is a simple markup structure — initialize it with one line of JS below.

<div class="timeline" data-mode="horizontal">
  <div class="timeline__wrap">
    <div class="timeline__items">
      <div class="timeline__item">
        <div class="timeline__content">
          <h5>2001</h5>
          <p>Example event content.</p>
        </div>
      </div>
      <!-- more items -->
    </div>
  </div>
</div>

2. Initialize with Vanilla JS (one line)

<script src="/path/to/timeline.min.js"></script>
<script>
  // initialize all timelines on the page
  timeline(document.querySelectorAll('.timeline'));
</script>

3. Initialize with jQuery (one line)

<script src="https://cdn.jsdelivr.net/npm/jquery@3.7.1/dist/jquery.min.js"></script>
<script src="/path/to/timeline.min.js"></script>
<script>
  // initialize using jQuery plugin
  jQuery(function () {
    $('.timeline').timeline();
  });
</script>

4. Local dev vs CDN

When developing locally, use the `dist/` files from the repo (run `npm run build`). For production, prefer the CDN snippet.
