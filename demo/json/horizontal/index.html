Creating the Horizontal Timeline:

Include links to CSS/Javascript files in your HTML document from the CDN
Add code for the timeline HTML elements
Create a .json file which contains your timeline data
Add a data attribute to HTML element with Path to your JSON file should be relative to the HTML document containing the timeline or, an absolute path path to your JSON file
<!-- This goes on the parent element which has class="timeline" -->

<div class="timeline" data-json-config="/path/to/your/data/file.json">
 ···
</div>
Save the file and load your HTML page in a browser
